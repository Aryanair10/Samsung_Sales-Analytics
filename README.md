# Samsung Supply Chain & Sales Analytics Dashboard

## Overview
Developed an interactive Power BI dashboard to analyze Samsung’s business performance across Sales, Production, Inventory, Procurement, and Shipment operations. The dashboard provides real-time insights into revenue trends, production efficiency, inventory risks, supplier performance, and logistics operations.

## Data Model
Designed using Star Schema:

### Dimension Tables
- Dim Date
- Dim Product
- Dim Customer
- Dim Supplier
- Dim Facility

### Fact Tables
- Fact Sales
- Fact Production
- Fact Inventory
- Fact Procurement
- Fact Shipment

## Key Insights
- Total Net Revenue reached 392.96M
- Total Profit was 94.50M
- Profit Margin was 24.05%
- Smartphones generated the highest revenue
- Retailer channel contributed the highest sales share
- Total Production reached 23M units
- Delayed Shipments were 1947
- Procurement Cost reached 610.35M
